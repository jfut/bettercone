## 💡 Using These Docs

### With Cursor
Cursor reads `.cursorrules` automatically. Reference these docs when needed:
```
@context.md Show me authentication patterns
@feature-templates.md Use CRUD template
```

### With Claude/ChatGPT
Start your session with:
```
Read the llms.txt file from BetterCone.
I want to add [YOUR FEATURE].
```

Then reference specific prompts from `AI_PROMPTS.md`.

### With v0.dev / Lovable
Copy prompts from `AI_PROMPTS.md` and specify:
- Use shadcn/ui components only
- Follow BetterCone patterns
