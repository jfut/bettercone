# InvoiceHistoryCard

The `InvoiceHistoryCard` component displays a link to view invoices via the Stripe Customer Portal.

<Callout type="info">
  This component is backend-agnostic and uses a callback to navigate to your Stripe portal implementation.
</Callout>

## Installation

```tsx
import { InvoiceHistoryCard } from '@bettercone/ui';
```

## Features

- ✅ **Backend-Agnostic** - Works with any Better Auth backend (Convex, Prisma, Supabase, Drizzle)
- ✅ **Stripe Portal** - Navigate to Stripe portal to view invoices
- ✅ **Status Badges** - Visual indicators for invoices found
- ✅ **Empty State** - Helpful message when no invoices exist
- ✅ **Configurable Limit** - Control max invoices display
- ✅ **Localization** - Full i18n support
- ✅ **Skeleton Loading** - Smooth loading states

## Invoice Data Structure

Each invoice contains:

```typescript
interface Invoice {
  id: string;              // Stripe invoice ID
  number: string;          // Invoice number (e.g., "INV-001")
  amountPaid: number;      // Amount in cents
  currency: string;        // Currency code (usd, eur, etc.)
  status: string;          // paid, open, void, uncollectible
  created: number;         // Unix timestamp
  hostedInvoiceUrl: string; // Stripe-hosted invoice page
  invoicePdf: string;      // Direct PDF download link
}
```

## Invoice Status

Different statuses with visual indicators:

| Status | Badge | Description |
|--------|-------|-------------|
| `paid` | Green | Successfully paid |
| `open` | Yellow | Awaiting payment |
| `draft` | Gray | Draft invoice, not finalized |
| `void` | Gray | Canceled/voided invoice |
| `uncollectible` | Red | Marked as uncollectible |

```tsx
// Status badge colors
const statusColors = {
  paid: "bg-green-100 text-green-800",
  open: "bg-yellow-100 text-yellow-800",
  void: "bg-gray-100 text-gray-800",
  uncollectible: "bg-red-100 text-red-800",
};
```

## Date Formatting

The component formats invoice dates for readability:

```typescript
// Format Unix timestamp to readable date
const formatDate = (timestamp: number) => {
  return new Date(timestamp * 1000).toLocaleDateString('en-US', {
    year: 'numeric',
    month: 'short',
    day: 'numeric',
  });
};

// Example output: "Oct 15, 2025"
```

## Currency Formatting

Amounts are formatted based on currency:

```typescript
const formatAmount = (amount: number, currency: string) => {
  return new Intl.NumberFormat('en-US', {
    style: 'currency',
    currency: currency.toUpperCase(),
  }).format(amount / 100); // Convert cents to dollars
};

// Examples:
// formatAmount(4999, 'usd') => "$49.99"
// formatAmount(4999, 'eur') => "€49.99"
```

## Customization

### Custom Invoice Display


## Notes

<Callout type="info">
  Invoices are automatically generated by Stripe when payments are processed. The component displays invoices from the active subscription only.
</Callout>

- Invoices sorted by date (most recent first)
- PDF download links valid for 30 days (Stripe default)
- Failed invoices remain visible for retry
- Voided invoices are kept for record-keeping
- All amounts include tax when applicable
