# PricingCard

The `PricingCard` component displays an individual pricing plan with features, pricing details, and subscription actions. It integrates with Stripe Checkout for seamless plan upgrades.

<Callout type="info">
  This component is designed to work within the PricingDashboard but can also be used standalone for custom pricing layouts.
</Callout>

## Installation

```tsx
import { PricingCard } from '@bettercone/ui';
```

## Features

- ✅ **Stripe Checkout** - Direct integration with Stripe subscription checkout
- ✅ **Organization Support** - Team plans require organization selection
- ✅ **Billing Intervals** - Monthly/yearly pricing with savings display
- ✅ **Feature List** - Checkmark list of plan features
- ✅ **Popular Badge** - Highlight recommended plans
- ✅ **Custom Styling** - Granular control over card appearance
- ✅ **Localization** - Full i18n support
- ✅ **Loading States** - Disabled state during checkout process
- ✅ **Type-Safe** - Full TypeScript with PricingPlan type

## PricingPlan Type

```typescript
interface PricingPlan {
  id: string;                    // Plan identifier (free, pro, team)
  name: string;                  // Display name
  description: string;           // Short description
  monthlyPrice: number;          // Monthly price in dollars
  yearlyPrice: number;           // Yearly price in dollars
  currency: string;              // Currency code (usd, eur, etc.)
  stripePriceIdMonthly: string;  // Stripe price ID for monthly billing
  stripePriceIdYearly: string;   // Stripe price ID for yearly billing
  features: string[];            // List of features
  popular?: boolean;             // Show "Popular" badge
  requiresOrganization?: boolean; // Team plans require org
  minSeats?: number;             // Minimum seats for team plans
  maxSeats?: number;             // Maximum seats for team plans
  trialDays?: number;            // Free trial period in days
}
```

## Savings Calculation

The component automatically calculates yearly savings:

```typescript
// Savings: (monthlyPrice * 12) - yearlyPrice
const savings = (29 * 12) - 290; // $58/year saved

// Display: "Save $58/year"
```

## Trial Period


## Checkout Integration

The component triggers Stripe Checkout when subscribed:

```tsx
const handleSubscribe = async (plan: PricingPlan) => {
  const priceId = billingInterval === 'month' 
    ? plan.stripePriceIdMonthly 
    : plan.stripePriceIdYearly;

  const result = await authClient.subscription.createCheckoutSession({
    priceId,
    successUrl: `${window.location.origin}/billing?success=true`,
    cancelUrl: `${window.location.origin}/pricing`,
    organizationId: selectedOrgId, // For team plans
  });

  if (result.data?.url) {
    window.location.href = result.data.url;
  }
};
```

<Callout type="warning">
  Stripe Checkout must be configured in your Stripe Dashboard with success/cancel URLs before subscriptions work.
</Callout>

## Customization

### Custom Price Formatting

```tsx
<PricingCard
  plan={plan}
  billingInterval="month"
  onSubscribe={handleSubscribe}
  formatPrice={(price, interval) => {
    if (price === 0) return 'Free';
    return `€${price}${interval === 'month' ? '/mo' : '/yr'}`;
  }}
/>
```

### Custom Savings Display

```tsx
<PricingCard
  plan={plan}
  billingInterval="year"
  onSubscribe={handleSubscribe}
  getSavingsText={(monthlyPrice) => {
    const savings = (monthlyPrice * 12) - plan.yearlyPrice;
    const percentage = Math.round((savings / (monthlyPrice * 12)) * 100);
    return `${percentage}% off`;
  }}
/>
```

## Notes

- Free plans (price = 0) show "Get Started" instead of pricing
- Popular badge is shown based on `popular` prop
- Team plans show seat range (e.g., "per user, 3-50 seats")
- Organization selector appears only when `requiresOrganization` is true
- Loading state disables button and shows spinner
