# Better Auth Plugins

Plugins extend Better Auth with additional functionality like usage tracking, rate limiting, and more.

## Official BetterCone Plugins

### Usage Tracking Plugin

Track API usage, enforce limits, and manage feature access at the organization level.

- **Package**: `@bettercone/better-auth-plugin-usage-tracking`
- **npm**: [View on npm](https://www.npmjs.com/package/@bettercone/better-auth-plugin-usage-tracking)
- **GitHub**: [View source](https://github.com/vncsleal/bettercone)

**Features**:
- 🎯 Automatic API tracking per organization
- 📊 Usage history for analytics
- 🔄 Flexible reset modes (lazy/scheduled/manual)
- 🚨 Smart callbacks (limit exceeded, 80% warning)
- 💰 Plan and features tracking
- 📈 Historical data with time-series
- 🔒 Secure with session auth + API key for cron
- 📦 Full TypeScript support

[Read full documentation →](/docs/plugins/usage-tracking)
